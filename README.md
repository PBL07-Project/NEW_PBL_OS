to run this code 

1 - create a empty folder
2 - git clone https://github.com/PBL07-Project/NEW_PBL_OS.git
3 - cd client
4 - npm install
5 - cd ..
6 - cd server
7 - npm install
8 - cd ..

now to run both client and server

in one terminal 
1 - cd client
2 - npm start

one another terminal in same root folder
1 - cd server
2 - npm start
